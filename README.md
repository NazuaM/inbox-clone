# Inbox Clone

A simple React + Vite application simulating an email inbox.

![App Screenshot](src\assets\demo.png) <!-- replace with your screenshot path -->

## Features
- Add emails with sender, subject, and status (sent/failed)
- Display emails in a list format
- Basic state management using React hooks
- Two versions: practice and working component

## Tech Stack
- **Frontend Framework:** React 19.1.1
- **Build Tool:** Vite 7.1.2
- **Language:** JavaScript (JSX), HTML, CSS
- **Package Manager:** npm
- **State Management:** React Hooks (`useState`)
